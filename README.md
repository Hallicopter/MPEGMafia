# MPEGMafia

This is a very early stage command line music/podcast player made in racket. 

### Goals
- [x] Play music.
- [x] Pause music.
- [x] Forward music
- [x] Rewind music (little hacky)
- [x] Progress bar.
- [ ] Easy install (WIP)
- [ ] Seconds counter (WIP)
- [ ] Skip to timestamp.
- [ ] Podcast RSS integraiton.
- [ ] MP3 support.
